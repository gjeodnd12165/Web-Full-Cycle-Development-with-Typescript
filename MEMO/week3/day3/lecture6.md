### URL + method
#### URL 연습
1. http://localhost:8888/postproduct -> "POST" /product
2. http://localhost:8888/select_all_products -> "GET" /products
3. http://localhost:8888/DeleteAllProducts -> "DELETE" /products
#### REST API 규칙
- 대문자 X, 소문자 O
- 언더바(_) X, 하이픈(-) O
- 마지막에 슬래시(/) 포함 X
- **행위를 포함하지 않는다 = 목적을 포함하지 않는다**
- 파일 확장자 포함 X
- 복수형을 쓴다
