### 느낀 점
리액트는 최근에 배워서 기억이 잘 난다. 템플릿으로 새 앱 만드는 것은 기억이 나지 않아서 매번 찾아보지만, 상관 없을 것 같다.  
가상 DOM 등 react의 동작 원리에 대해서는 자세히 알아본 적이 없어서 흥미로웠다. 나중에 이것을 깊게 이해해야할 떄가 있을 것 같다.
