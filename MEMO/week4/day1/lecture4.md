### 자바스크립트 객체, 배열 비구조화
#### 객체의 비구조화
```js
const obj = {
  v: "value",
  t: "time"
}
const { v, t } = obj;
// v = "value"
// t = "time"
```
다음과 같이 객체 내의 값을 바로 꺼내 쓸 수 있다.  
여기서는 변수의 이름을 key 이름과 맞추지 않으면 `undefined`가 된다.
#### 배열의 비구조화
```js
const array = [1, 2, 3, 4, 5];
const [ , num2, num3, , num5];
// num2 === 2
// num3 === 3
// num5 === 5
```
다음과 같이 배열 내의 값을 바로 꺼내 쓸 수 있다.  
여기서는 순서를 맞추어야하며, 순서를 건너뛰고 싶으면 그 자리를 비워 놓아야한다.
