### 로그인 세션 - 인증과 인가
#### 로그인 세션 만료
기본적으로 유저 정보의 보안 때문이다. 로그인이 평생 되어있다면, 로그아웃을 까먹었을 때 정보가 새어나갈 수 있을 것이다.  
#### 인증과 인가
- 인증 (=로그인)  
Authentication  
유저의 각각의 정보를 다룰 필요가 있을 떄
- 인가  
Authorization  
: e.g. 같은 사이트에서 관리자 / 고객에 따라 접근할 수 있는 페이지가 다르다 
> 관리자든 고객이든 **인증**을 통해서 사이트에 가입된 사용자라는 것을 증명하는것  
인증 후에, 접근 권한을 살펴보는 것을 **인가**
